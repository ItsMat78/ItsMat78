## Things I've built

**[Lagrange-Lock](https://github.com/ItsMat78/lagrange-lock):** I got curious whether a learning agent could hold a satellite steady near a Lagrange point, where the orbit is naturally unstable. Turns out yes. Built with [Sameer](https://github.com/Sameerkc3579/) & supervised by [Dr. Avantika Singh](https://iiitnr.ac.in/content/avantika-singh). Want to [test it?](https://lagrange-lock.shreyashrai.com)  
 `Python, PyTorch, Numba, Three.js` 

**[journal-advisory](https://github.com/ItsMat78/journal-advisory):** A retrieval-augmented TUI over my own Obsidian journal. I use it to ask questions about my life, without losing context.  
`sentence-transformers, ChromaDB, Python`

**[Taxshila Companion](https://github.com/ItsMat78/taxshila-companion):** Institutional management app, built solo and end to end. Live on the Play Store with real daily users and real database traffic. This one started as a problem my family's study-hall business had. Solved. [Here's the deployment.](https://taxshilacompanion.vercel.app/home)  
`Next.js, TypeScript, Firebase`

**[FDTD simulation using DMD Method](https://github.com/ItsMat78/FDTD):** Under the guidance of [Dr. Maifuz Ali](https://iiitnr.ac.in/content/maifuz-ali), I modeled 1D FDTD electromagnetic wave propagation and applied Dynamic Mode Decomposition (DMD) to predict field evolution. Cut computation ∼10x versus direct simulation while maintaining ~96% accuracy.  
`Numpy, DMD`
